Sample Database Structure.

This is an overview of sample database that you imported from setup post.

Overview

We’re assuming that we have an eCommerce website and have some tables like, users, products, orders, and users_log.

Users table

Users table have the following structure.

columntype
idint
datetime_joineddatetime
first_namevarchar
last_namevarchar
emailvarchar
usernamevarchar
activeboolean

Products table

Products table have the following structure.

columntype
idint
created_atdatetime
namevarchar
priceint
categoryforeign key
descriptionvarchar
sale_priceint / nullable
